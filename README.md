# VT Styles 🖥️

A desktop application built with **Windows Forms (C#)** and **inbuilt SQL database**.  
This project demonstrates a complete salon/hairstyle management system with multiple forms, admin panel, and booking features.

---

## 🚀 Features
- **User Management**: Registration, login, and profile handling
- **Admin Panel**: Manage admins, credentials, and salon operations
- **Booking System**: Create and manage appointments
- **Style Catalog**: Forms showcasing different hairstyles (beard, bangs, mullet, mustache, etc.)
- **Database Integration**: Local SQL database for persistent storage
- **Executable Build**: Ready‑to‑run `VT STYLES.exe`

---

## 📂 Project Structure
Some key files included in the repository:

- **Forms**
  - `Form1.cs`, `Form2.cs` – Main entry forms
  - `adminlogin.cs`, `adminpannel.cs`, `manageadmin.cs` – Admin management
  - `booking.cs`, `app.cs` – Booking and application logic
  - Hairstyle forms: `fullbeard.cs`, `fullmullet.cs`, `fullmustache.cs`, `fullspike.cs`, etc.
  - `home.cs`, `men.cs` – Navigation and user interface

- **Resources**
  - `.Designer.cs` – UI design code
  - `.resx` – Resource files for each form
  - `Database.mdf` – Inbuilt SQL database file

- **Executables**
  - `VT STYLES.exe` – Compiled application
  - `VT STYLES.vshost.exe` – Debug host
